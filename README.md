#  Mis UserScripts para Tampermonkey

Repositorio personal para gestionar scripts de personalización web.

## Scripts Disponibles

### 1. Claude AI - Mostrar Solo Proyectos
Optimiza la interfaz lateral de Claude.ai para centrarse en el flujo de Proyectos.

* **[Instalar Script](https://raw.githubusercontent.com/rx32555/tampermonkey-userscripts/main/claude/claude_ocultar_panel.user.js)** (Requiere Tampermonkey)
* **Características:** Oculta secciones de "Recientes", "Nuevo Chat" y "Artefactos".

---
## Notas de Uso
- Los scripts se actualizan automáticamente al subir una nueva versión a la rama `main`.
- Ideal para compartir "Proyectos" con misma cuenta Claude.ai con colegas de confianza, mostrando solo elementos básicos en la UI.
